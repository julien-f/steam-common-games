Steam Common Games

Find games shared across multiple Steam libraries, with ratings and playtime estimates

Players

One player: browse their full library. Multiple players: find games everyone owns.

How it works

  1. Add a player — enter any Steam username, profile URL, or 64-bit ID
  2. Add more players to compare — or keep just one to browse a single library
  3. Use + Steam Family — adds a shared-library account to the same slot; their games are merged before comparing
